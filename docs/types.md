# 类型系统

LongLang 提供了丰富的基础类型系统，支持多种整数、浮点数、布尔和字符串类型。

## 类型一览表

### 整数类型

| 类型 | 描述 | 字节数 | 取值范围 |
|------|------|--------|----------|
| `int` | 有符号整型，与平台 CPU 位宽一致 | 8 (64位平台) | -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807 |
| `i8` | 8位有符号整型 | 1 | -128 ~ 127 |
| `i16` | 16位有符号整型 | 2 | -32,768 ~ 32,767 |
| `i32` | 32位有符号整型 | 4 | -2,147,483,648 ~ 2,147,483,647 |
| `i64` | 64位有符号整型 | 8 | -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807 |

### 无符号整数类型

| 类型 | 描述 | 字节数 | 取值范围 |
|------|------|--------|----------|
| `uint` | 无符号整型，与平台 CPU 位宽一致 | 8 (64位平台) | 0 ~ 18,446,744,073,709,551,615 |
| `u8` | 8位无符号整型 | 1 | 0 ~ 255 |
| `u16` | 16位无符号整型 | 2 | 0 ~ 65,535 |
| `u32` | 32位无符号整型 | 4 | 0 ~ 4,294,967,295 |
| `u64` | 64位无符号整型 | 8 | 0 ~ 18,446,744,073,709,551,615 |

### 浮点数类型

| 类型 | 描述 | 字节数 | 精度 |
|------|------|--------|------|
| `float` | 浮点数，与平台 CPU 位宽一致 | 8 (64位平台) | 约15-17位有效数字 |
| `f32` | 32位单精度浮点数 | 4 | 约6-9位有效数字 |
| `f64` | 64位双精度浮点数 | 8 | 约15-17位有效数字 |

### 其他基础类型

| 类型 | 描述 | 字节数 | 说明 |
|------|------|--------|------|
| `bool` | 布尔类型 | 1 | 值为 `true` 或 `false` |
| `string` | 字符串类型 | 可变 | UTF-8 编码字符串 |
| `any` | 任意类型 | 可变 | 特殊联合类型，可以存储任意类型的值 |
| `void` | 无类型 | 0 | 表示函数无返回值 |

## 字面量语法

### 整数字面量

```longlang
a := 42          // 十进制整数
b := -100        // 负整数
c := 0           // 零
```

### 浮点数字面量

```longlang
pi := 3.14159    // 浮点数
e := 2.71828     // 浮点数
temp := -40.5    // 负浮点数
```

### 布尔字面量

```longlang
yes := true      // 真
no := false      // 假
```

### 字符串字面量

LongLang 支持三种字符串声明方式：

```longlang
// 双引号字符串（支持转义字符）
s1 := "Hello, World!"
s2 := "Line1\nLine2"    // 包含换行

// 单引号字符串（支持转义字符）
s3 := 'Hello, World!'
s4 := 'It\'s OK'        // 转义单引号

// 反引号原始字符串（不处理转义字符）
s5 := `原始字符串，\n 不会被转义`
s6 := `可以包含
多行
内容`
```

### null 值

```longlang
var x any
x = null             // 设置为空值
```

## 类型转换

整数和浮点数可以混合运算，结果为浮点数：

```longlang
a := 10          // int
b := 3.5         // float
c := a + b       // 结果为 13.5 (float)
d := a * b       // 结果为 35.0 (float)
```

## 类型声明

### 变量类型声明

```longlang
var name string = "Alice"
var age int = 25
var height float = 1.75
var isActive bool = true
```

### 函数参数类型声明

```longlang
fn add(a:int, b:int) int {
    return a + b
}

fn multiply(x:float, y:float) float {
    return x * y
}

fn greet(name:string) string {
    return "Hello, " + name
}
```

### 类成员变量类型

```longlang
class Person {
    public name string
    public age int
    public height float
    public isActive bool
}
```

## 特殊类型

### any 类型

`any` 是一个特殊的联合类型，可以存储任意类型的值：

```longlang
fn printAny(value:any) {
    fmt.Println(value)
}

printAny(42)         // 整数
printAny("hello")    // 字符串
printAny(3.14)       // 浮点数
printAny(true)       // 布尔值
```

### void 类型

`void` 用于表示函数没有返回值：

```longlang
fn doSomething() void {
    fmt.Println("执行操作")
    // 无需 return
}
```

## 类型总结

| 分类 | 类型 | 用途 |
|------|------|------|
| 整数 | `int`, `i8`, `i16`, `i32`, `i64` | 存储整数，有符号 |
| 无符号整数 | `uint`, `u8`, `u16`, `u32`, `u64` | 存储非负整数 |
| 浮点数 | `float`, `f32`, `f64` | 存储小数 |
| 布尔 | `bool` | 逻辑判断 |
| 字符串 | `string` | 文本数据 |
| 任意 | `any` | 动态类型 |
| 空 | `void`, `null` | 无值/空值 |
